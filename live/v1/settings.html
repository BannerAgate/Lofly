Google OAuth Refresh Token

  1. Ga naar Google Cloud Console
  2. Maak een OAuth 2.0 client aan (type: Web application)
  3. Voeg https://developers.google.com/oauthplayground toe als redirect URI
  4. Ga naar OAuth Playground
  5. Klik op het instellingen-icoontje → vink "Use your own OAuth credentials" aan
  6. Vul je Client ID en Secret in
  7. Zoek naar https://www.googleapis.com/auth/business.manage en autoriseer
  8. Klik "Exchange authorization code for tokens"
  9. Kopieer de Refresh token en plak hem hierboven
